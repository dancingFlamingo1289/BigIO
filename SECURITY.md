# Security Policy

## Supported Versions
Currently, only the latest version of BigIO is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability
I take the security of this library seriously. If you find a security vulnerability (e.g., path traversal in the Registry, memory leaks, or unauthorized file access), please do not open a public issue.

Instead, please report it directly to the maintainer:
- **Contact**: Elias Kassas
- **Process**: Please provide a detailed description of the vulnerability and a proof of concept if possible.

I will acknowledge your report within 48 hours and work on a fix as quickly as possible.
